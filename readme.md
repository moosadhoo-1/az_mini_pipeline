# Data Engineering Pipeline (Bronze --> Silver --> Gold)

## Overview
Simple data engineering pipeline built with Python. 
It shows how raw/dirty data is cleaned and then aggregated for
business insights, Data Science etc.

## Architecture
CSV files containing raw transaction and user data.
After that the Data gets transformated in standardized text formats
and invalid records removed.
Aggregation in the end: Total transactions per day and Number of transactions per day.

## Features
End-to-end ETL Pipeline
Data Cleaning
Aggregation
Medallion Architecture

## Tech Stack
- Python (pandas)
- Git
